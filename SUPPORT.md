# Support

Need help with ClanSocket? Here's where to look.

## Using ClanSocket

- **Website & dashboard** — [clansocket.com](https://clansocket.com).
- **RuneLite plugin** — install **ClanSocket** from the Plugin Hub
  (`Configure → Plugin Hub → search "ClanSocket"`). For setup and claiming your
  clan, see the
  [plugin wiki](https://github.com/osrs-clansocket/clansocket-plugin/wiki).

## Questions & problems

- **Something's broken?** Open a bug report on the relevant repository.
- **Want a feature?** Open a feature request.
- **A "how do I…" question?** Use the repository's Discussions if enabled,
  otherwise open an issue.

Please search existing issues first — your question may already be answered.

## Security

Do **not** report security vulnerabilities in public issues. Follow
[SECURITY.md](SECURITY.md) instead.
